# CV Files Directory

This directory contains your CV files in PDF format.

## Required Files

- `CV_EN.pdf` - Your CV in English
- `CV_ES.pdf` - Your CV in Spanish

## Instructions

1. Place your English CV as `CV_EN.pdf` in this directory
2. Place your Spanish CV as `CV_ES.pdf` in this directory
3. The website will automatically download the appropriate file based on the selected language

## Notes

- Make sure your PDF files are optimized for web viewing
- Recommended file size: under 2MB for faster downloads
- Use consistent formatting between both language versions
